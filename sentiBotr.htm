SENTIBOTR


A Twitter bot that analyses and tweets the sentiment of trending topics in the UK.
View On Github.

Tweeting since 28/10/14.
Note: AdBlockers can hide the Twitter timeline above.

Dependencies

  • Twitter4j Library
  • sentiment140 API

Configuration

To configure sentiBotr for use, add your Twitter OAuth credentials to the twitter4j.properties file. This will allow twitter4j to connect to the Twitter API.

You can further configure the bot by changing the values in the constants interface.

Hosting

Currently, I am hosting my bot on an Amazon Web Services Linux micro instance. The bash script below runs the jar, and is executed every hour by a cron job.
