SimpleDo

A simple, open source 'To-Do' application for Android, with an emphasis on minimalism. Available free, for Android 4.0.1 and up.


Download

GitHub

Features


Reminders

Add Reminders so no task is forgotten.

Due Dates

Set Due Dates so no deadline is missed.

Groups

Seperate personal tasks from work.

Priority

Set a tasks priority (Low, Medium or High).

Filters

Filter tasks by several parameters.

Calendar Integration

Adds tasks to your phones native calendar.
